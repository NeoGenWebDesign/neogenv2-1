
# neogen web design

Modern, clean, and agentic web design and development, based in South Africa.

## Features

- Modern, clean, and anti-design inspired UI
- About, social/contact, and company info
- Agentic UX: Chatbud (intelligent assistant), predictive personalization
- High-fidelity interactivity: WebGPU/WebGL, physics-based motion, scrollytelling, kinetic typography
- Bento grid layout, tactile maximalism, brat aesthetic
- Under the hood: WebAssembly (WASM), PWA, sustainable web design, spatial audio

## Getting Started

### Development

```bash
npm install
npm run dev
```

Visit [http://localhost:5173/](http://localhost:5173/) to view the site.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

Edit `src/App.tsx` and `src/App.css` to update content and styles.
